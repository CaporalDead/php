Les sessions et cookies

Présentation

Les sessions et les cookies permettent d’enregistrer des données spécifiques à un internaute de façon temporaire.

Cookies Session
Stocké sur l’ordinateur de l’internaute Stocké sur le serveur
Supprimé à une date fixe Supprimé à la fin de la session (fermeture du navigateur)

Les cookies

Pour définir la valeur d’un cookie, on utilise la fonction PHP setcookie().

Pour récupérer la valeur d’un cookie, on utilise la superglobale $_COOKIE

setcookie('nom', 'Dupont', time() + 3600); // Création d'un cookie ayant une durée de vie d'une heure
setcookie('prenom', 'Jean', time() + 3600*24); // Création d'un cookie ayant une durée de vie d'un jour

echo 'Bonjour ' . $_COOKIE['nom'];
La fonction setcookie doit impérativement être utilisée avant le code HTML !

Les cookies

Pour supprimer un cookie, il faut définir sa date d’expiration à une date antérieur. On peut également utiliser la fonction PHP unset() pour supprimer la variable.

setcookie('nom', null, time() - 3600);
setcookie('prenom', null, time() - 3600);
unset($_COOKIE['nom']);
unset($_COOKIE['prenom']);

Les sessions

Pour obtenir ou définir la valeur d’une session, on utilise la fonction PHP session_start() puis on accède à la superglobale $_SESSION. La fonction session_destroy() permet de supprimer toutes les valeurs associées à une session.

session_start();
$_SESSION['nom'] = 'Dupont';
$_SESSION['prenom'] = 'Jean';
La fonction session_start doit impérativement être utilisée avant le code HTML !

Les sessions

Pour supprimer une session, on utilise la fonction PHP session_destroy(). Pour pouvoir utiliser cette fonction, il faut avoir démarrer la session avec la fonction session_start().

session_start();
session_destroy();
