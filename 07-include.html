Structure d'une page

HEADER
MENU
BODY
FOOTER

Avantages

Le code du header, du menu et du footer est copié sur l’ensemble des pages.

Solution : Ségmenter le code source en plusieurs fichiers

  • Alléger le code source
  • Améliorer la maintenabilité du code
  • Code source réutilisable
  • Séparer les paramètres du site

Les fonctions

Fonction Description
include L'instruction de langage include inclut et exécute le fichier spécifié en argument.
include_once Le comportement est similaire à include, mais la différence est que si le code a déjà été inclus, il ne le sera pas une seconde fois.
require require est identique à include mise à part le fait que lorsqu'une erreur survient, il produit également une erreur fatale.
require_once L'instruction require_once est identique à require mis à part que PHP vérifie si le fichier a déjà été inclus, et si c'est le cas, ne l'inclut pas une deuxième fois.
